\subsection{Operation Model for addQuestion}

\label{OM-addQuestion}


The \msrcode{addQuestion} operation has the following properties:

	\begin{operationmodel}
	\addheading{Operation}
	\adddoublerow{addQuestion}{Add new questions
	 }

	\addrowheading{Parameters}
	\addnumbereddoublerow{}{question: dtQuestion}{New Question} 

	\addrowheading{Return type}
	\addsinglerow{ptInteger}

		

	\addrowheading{Post-Condition (functional)}
	\addnumberedsinglerow{PostF}{return key - size of questions }

	\end{operationmodel}

	% MCL Listing

	postF{if

	/* Post F01 */
	let Self:ctAdminQuestions in

	)
	then (result = true)
	else (result = false)
	endif}
